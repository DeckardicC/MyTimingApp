# MyTiming

**Версия 1.0.0** - Стабильная рабочая версия

Мобильное приложение для генерации таймингов праздничных мероприятий.

## Описание

MyTiming — это Flutter-приложение, предназначенное для автоматической генерации расписаний (таймингов) для детских праздников и мероприятий. Приложение позволяет создавать структурированные планы мероприятий с учетом различных типов программ и локаций.

**Версия 1.0.0** — это полноценная рабочая версия приложения со всеми основными функциями. Все модули протестированы и готовы к использованию.

### Основные возможности

- **Генерация таймингов для VR-центров** (Another World, Ели Пели)
  - Поддержка различных пакетов: Мини, Оптимал, Макси, Стандарт, VIP
  - Автоматический расчет времени для каждого этапа мероприятия

- **Генерация таймингов для КХ/Космобаза**
  - Поддержка различных залов (Большой зал, Малый зал)
  - Гибкая настройка программ с учетом типа мероприятия
  - Выбор времени приема пищи (в начале или после игры)

- **Удобный интерфейс**
  - Современный Material Design 3
  - Интуитивная навигация
  - Копирование готовых таймингов

## Установка и сборка

### 1. Клонирование репозитория

```bash
git clone https://github.com/DeckardicC/MyTimingApp.git
cd MyTimingApp
```

Или скачайте архив релиза версии 1.0.0 с GitHub.

### 2. Установка зависимостей

```bash
flutter pub get
```

### 3. Сборка приложения

#### Для Android (APK)

**Автоматическая сборка:**
```bash
./build_apk.sh
```

Скрипт автоматически найдет Flutter SDK, установит зависимости и соберет релизный APK. Готовый файл будет находиться в корне проекта как `app-release.apk`.

**Ручная сборка:**
```bash
flutter build apk --release
```

APK файл будет находиться в `build/app/outputs/flutter-apk/app-release.apk`.

#### Для iOS (требуется macOS и Xcode)

```bash
flutter build ios --release
```

### 4. Установка на устройство

#### Android

1. **Через ADB (Android Debug Bridge):**
   ```bash
   adb install app-release.apk
   ```

2. **Через файловый менеджер:**
   - Скопируйте `app-release.apk` на Android-устройство
   - Откройте файл на устройстве
   - Разрешите установку из неизвестных источников (если требуется)
   - Установите приложение

3. **Через USB:**
   ```bash
   flutter install
   ```
   (устройство должно быть подключено и включена отладка по USB)

#### iOS

1. Откройте проект в Xcode:
   ```bash
   open ios/Runner.xcworkspace
   ```

2. Подключите iOS-устройство к Mac

3. Выберите устройство в Xcode и нажмите "Run"

## Структура проекта

```
lib/
├── main.dart              # Точка входа приложения
├── main_screen.dart       # Главный экран с выбором типа тайминга
├── vr_page.dart          # Экран генерации таймингов для VR-центров
├── kh_page.dart          # Экран генерации таймингов для КХ/Космобаза
├── timing_generator.dart # Генератор таймингов для VR
└── kh_timing_generator.dart # Генератор таймингов для КХ/Космобаза
```

## Разработка

### Запуск в режиме разработки

```bash
flutter run
```

### Запуск на конкретном устройстве

```bash
flutter devices                    # Список доступных устройств
flutter run -d <device_id>         # Запуск на конкретном устройстве
```

### Генерация иконки приложения

1. Поместите PNG файл 1024x1024 пикселей в `assets/icon/icon.png`
2. Запустите:
   ```bash
   flutter pub run flutter_launcher_icons
   ```
